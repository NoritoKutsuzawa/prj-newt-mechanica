# prj-newt-mechanica

ブラウザゲームをまとめるためのリポジトリです。  
`index.html` をトップページにして、そこから各ゲームへ移動できます。

## 収録ゲーム

- Shooting Game: `games/shooting/index.html`
- Reflex Click Game: `games/reflex/index.html`

## ローカル確認

静的ファイルなので `index.html` をブラウザで開くだけで確認できます。

## Vercel 連携

このリポジトリは静的サイトとしてそのまま Vercel にデプロイできます。  
Vercel プロジェクトにリポジトリを接続し、Framework Preset は `Other` のままで動作します。
